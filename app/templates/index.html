FollowInsights

Available on github

Followinsights was highly inspired on the Open Source Report Card by Dan Foreman-Mackey.

Consists in getting the information about the people who are followed by an user and the people followed by they.

The idea is to find some users that maybe have some interesting code to share and are not followed by the user. It's not about find who is BETTER or MORE POPULAR, it's just about find some CURIOSITIES about the NETWORK where the user is inserted.

The application uses the dump from the GHTorrent project and will be outdated in most of the cases.
